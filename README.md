### Инструменты для работы с ДНК и текстовыми данными
1. `clean_genes.py` — очистка списка генов от лишних пробелов
2. `average_numbers.py` — сумма и среднее значение списка чисел
3. `motif_search.py` — поиск всех вхождений ATG в ДНК
4. `transcription.py` — транскрипция ДНК в РНК
5. `gene_analyzer.py` — чтение файла с генами и анализ длин
6. Motif Finder — поиск ДНК-мотивов в FASTA-файлах(cмотреть ниже более детальное описание)

## Motif Finder
Программа читает FASTA-файл, очищает последовательности от недопустимых символов (оставляет только A, T, G, C) и находит все позиции, где встречается заданный мотив (по умолчанию — ATG).

# Возможности
- Чтение файлов в формате FASTA
- Автоматическая очистка последовательностей (удаление символов, не являющихся A/T/G/C)
- Сообщение о том, какие символы были удалены
- Поиск всех вхождений мотива в каждой последовательности
- Пропуск пустых последовательностей
- Читаемый вывод с указанием заголовка и позиций
# Как запустить
1. Убедитесь, что у вас установлен Python 3.6 или выше.
2. Поместите FASTA-файл в ту же папку, что и программу, или укажите полный путь к нему.
3. Запустите программу.
