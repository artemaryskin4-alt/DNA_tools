# Инструменты для работы с ДНК и текстовыми данными
1. `clean_genes.py` — очистка списка генов от лишних пробелов
2. `average_numbers.py` — сумма и среднее значение списка чисел
3. `motif_search.py` — поиск всех вхождений ATG в ДНК
4. `transcription.py` — транскрипция ДНК в РНК
5. `gene_analyzer.py` — чтение файла с генами и анализ длин
